## Getting started

1. Install dependencies
```BASH
yarn install
```
2. Build plugin in development mode or run in watch mode
```BASH
yarn dev
```
or
```BASH
yarn watch
```
3. Build plugin in production mode
```BASH
yarn build
```
4. Start Postgres, OpenNMS Horizon and Grafana via Docker
```BASH
docker-compose up
```
